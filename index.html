Meine HTML / CSS / Javascript Sammlung

Diese Webseite entsteht im Zuge des udemy Kurses "50 HTML CSS JS PROJECTS FOR BEGINNERS" bzw. teilweise auch aus Anregungen aus anderen Kursen. Während ich den Kurs mache, wird der Inhalt dieser Webseite ständig erweitert (aber ich mache den Kurs nur von Zeit zu Zeit, darum kann es sein, dass es dauert, bis wieder etwas dazu kommt).

Das Grid mit den Projekten wird dynamisch über ein Javascript aufgebaut. Im Moment sind die "Daten" für die Projekte noch hartcodiert im Skript in einem JSON hinterlegt. In weiterer Folge soll das JSON aus einem File ausgelesen werden - und außerdem soll man ein neues Projekt mit Hilfe einer "make" Funktion dem JSON hinzufügen können.

Dinge, die ich unterwegs gelernt habe (leider habe ich erst bei Projekt 8 angefangen, dass mitzuschreiben) ...

  • padding-top bezieht sich auch auf width des übergeordnenten Elements, nicht auf height. Umgehung: position:absolute und top bzw. bottom verwenden (die beziehen sich erwartungsgemäß auf height)Abschnitt 8
  • In Javascript kann man nicht auf pseudo-Elemente zugreifen, weil sie nicht Teil des DOM sind. Wenn man sie also stylen will, muss man mit CSS-Variablen arbeiten (deren Wert man mit setProperty setzen kann)Abschnitt 11
